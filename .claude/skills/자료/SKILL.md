# 자료 — 무엇이든 만들기 (라우터)

무엇을 만들지 자연어로 적으면, 가장 알맞은 생성기를 골라 **그 자리에서** 만들어 줘요. 어떤 `/명령`을 써야 할지 모를 때 여기서 시작하세요.

## 동작 방식
1. `$ARGUMENTS`(사용자가 원하는 결과물)를 읽고, 아래 표에서 **가장 잘 맞는 모듈 하나**를 고르세요. (용도·키워드 참고)
2. 고른 모듈의 스킬 파일 `.claude/skills/<명령이름>/SKILL.md`을 **Read 도구로 읽고**, 그 "생성 지침"을 그대로 따라 결과를 만드세요. 사용자의 입력을 그 스킬의 입력으로 사용합니다.
3. 어떤 모듈인지 애매하면(후보가 둘 이상이면), 후보 2개만 제시하고 **한 번만** 물어본 뒤 진행하세요.
4. 다 만든 뒤에는 그 스킬과 동일하게 **파일 내보내기**(워드·한글·PPT)를 제안하세요.
5. 사용자가 이미 `/시험지`처럼 정확한 명령을 알고 있으면 그 명령을 직접 쓰는 게 더 빠르다고 한 줄로 알려 주세요.

## 라우팅 표

| 명령 | 용도 | 대표 키워드 |
|---|---|---|
| `/시험지` | 배점표·정답표·해설까지 있는 완성형 시험지 (중간·기말 대비) | 시험, 시험지, 모의고사, 중간고사 |
| `/학습지` | 한 단원·스킬 집중 연습지 (숙제·워밍업용, 간단한 형태) | 학습지, 연습지, 워크시트, 숙제 |
| `/퀴즈` | 빠르게 이해를 확인하는 짧은 퀴즈 (간단한 형태) | 퀴즈, 쪽지시험, 미니테스트, 형성평가 |
| `/단어장` | 단어 목록 → 단어장 (미국식 발음·예문과 해석 포함) | 단어장, 영단어, 어휘, 단어 정리 |
| `/PPT` | 발표용 슬라이드 구성 (장별 핵심 + 발표 노트) | PPT, 발표자료, 슬라이드, 프레젠테이션 |
| `/학습노트` | 핵심 개념 요약 노트 (정의·핵심·예시 중심) | 학습노트, 정리노트, 요약, 개념정리 |
| `/수업지도안` | 도입-전개-정리 수업지도안 (활동·시간 배분·평가 포함) | 수업지도안, 교안, 지도안, 수업계획 |
| `/이력서` | 학력·경력·강점을 날짜순으로 정리한 이력서 | 이력서, CV, resume |
| `/자기소개서` | 경험을 이야기로 풀어 강점을 보여주는 자기소개서 | 자기소개서, 자소서, cover letter |
| `/소설` | 장르·소재만 고르면 이야기 초고 완성 | 소설, 글쓰기, 이야기, 창작 |
| `/엑셀` | 수식·차트·데이터 정리를 한 번에 | 엑셀, 구글시트, 스프레드시트, 수식 |

> 표에 없는 요청(예: 번역, 단순 질문)은 굳이 한 모듈로 욱여넣지 말고, 일반 대화로 도와주세요.
