# 인쇄 품질 검증 보고서 (컬러 + 흑백 그레이스케일)

- 대상: WeasyPrint 렌더러 2종 — `scripts/exam_pdf.py`(B4 시험지, teacher/student/key 3변형),
  `scripts/doc_pdf.py`(A4, vocab/notes/quiz/worksheet/doc 5타입 × 7테마)
- 기준: **한국 학교의 기본은 흑백 레이저 인쇄.** 모든 "글자를 담는" 전경/배경 쌍이
  **컬러값과 그레이스케일 변환값(L = 0.299R+0.587G+0.114B, Pillow `'L'` 동일) 모두**
  WCAG 대비 4.5:1 이상이어야 하고, 괘선·테두리는 0.75pt 이상이어야 함.
- 방법: 실제 샘플(scratchpad `gen_s5/*.md`)로 16개 PDF 전수 렌더 →
  `pdftoppm` 래스터(150dpi 전면 + 300dpi 부분 크롭) → Pillow `'L'` 흑백 변환 →
  팔레트 헥스 기반 대비 전수 계산(검증 스크립트: scratchpad
  `printqa/contrast_check.py`) + 래스터 육안 확인 → 수정 → 재렌더·재계측(1회 패스로 전 항목 해소).
- 근거 이미지: scratchpad `printqa/png/{before,after}/`, 흑백은 `printqa/gray/{before,after}/`
  (커밋하지 않음). 검증일: 2026-07-02.

## 4. 물리 점검 결과

- **폰트 임베딩**: 16개 PDF 전수 `pdffonts` — **모두 emb=yes, 비임베드 0건**.
  퀴즈 콘텐츠의 한자(私商)는 번들 NotoSansKR에 글리프가 없어 `Noto Sans CJK KR`로
  폴백되지만, 이 폰트는 Dockerfile이 `fonts-noto-cjk`로 프로덕션 이미지에 설치하고
  렌더 시 서브셋 임베드됨(emb=yes) — 규칙 위반 아님. 로컬 검증 환경도 동일 구성.
- **흑백 판독(래스터 육안)**: OMR 기입(진네이비 원+흰 숫자) vs 미기입(중간 회색 번호)
  흑백에서 명확히 구분 — `gray/after/omr_bubbles_crop.png`. 표지 브라스 라벨(난이도·안내),
  ★Killer, 품사 칩, 정답 바 모두 판독 가능 — `gray/after/exam_cover_top_crop.png`,
  `gray/after/vocab_chips_crop.png`, `gray/after/quiz_green_p1.png`.
- **제목 위계**: 색이 아니라 크기·굵기·블록(바·카드)으로 구분됨 — 흑백 래스터에서 위계 유지 확인.
- **페이지 나눔**: exam teacher 9p 전수 — 해설 카드(`ex-card`)·문항(`item`)·배점표 행 분할 없음,
  파트 머리글이 첫 카드와 분리되지 않음(`break-after: avoid` 기존 동작 확인).
  doc 쪽은 표 행 분할 금지(`tr { break-inside: avoid }`)·`thead` 반복·제목-첫 문단 분리
  금지(h1~h3)를 이번에 추가(예방 조치, 페이지 수 변동 없음). 학습지·퀴즈 해설 페이지
  래스터 확인 — 카드 단위로 온전함.
- **빈 마지막 페이지**: 16개 PDF 전수 마지막 페이지 텍스트 추출 — **빈 페이지 0건**.
- **표지 목차 규칙**: `coverFitsPage1: true` 유지(teacher·student), QA 루프 1패스 클린.
- **페이지 수 불변**: teacher 9p·student 5p·key 1p, doc 전 타입·전 테마 동일 → 골든 재생성 불필요.

## 8. 범위 외 / 스킵

- **docx·pptx·xlsx**(클라이언트 Office 파일): 인쇄는 Office가 담당 — 과업 범위 외.
- **사이트 HTML**: 별도 과업.
- **표 격자선 #CBD2DA·구분선 #E5E7EB (doc)**: 두께 0.75pt 충족, 래스터에서 판독 가능 확인
  → 텍스트가 아니므로 4.5:1 비적용, 변경하지 않음(과보정 방지).
- **doc_pdf 제목 미지정 시 "문서" 표기, 일부 파트명 공백(P4)**: 입력/파서 이슈로 인쇄
  품질 범위 외 — 관찰만 기록.
