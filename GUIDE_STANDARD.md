# 초보자(어르신) 가이드 폼 표준

빈 입력창 대신 **질문에 답만 하면 결과가 완성되는** 단계별 폼의 공통 표준입니다.
PPT 모듈의 가이드 흐름을 구조 참고로 삼아 정의했고, 시험지·자기소개서 모듈에 먼저 적용했습니다.

## 원칙

1. **한 화면에 질문 하나** + 진행 단계 표시(예: `2 / 5` + 진행 막대).
2. **쉬운 말, 전문용어 금지.** 꼭 써야 하면 괄호로 풀이 — 예: "객관식(보기 중 고르기)".
3. **선택지는 한 단계에 3~4개 이하**, 각 선택지에 짧은 예시(`example`).
4. **"잘 모르겠어요"를 항상 제공** — 누르면 기본값(`skipValue`/`default`)으로 진행.
5. **큰 글씨·큰 버튼·넉넉한 터치 영역, 모바일 우선.**
6. **마지막에 "이렇게 만들게요" 요약**을 보여주고 확인받은 뒤 생성.
7. 기존 PPT 가이드 흐름을 참고하되 위 원칙을 우선.

## 구현 방식 (기존 구조 비파괴)

- 데이터: 모듈의 기존 `guide`/`options` 스키마를 그대로 쓰되, 필드에 `question`(큰
  질문) · `help`(쉬운 설명) · `skipValue`(모르겠어요 기본값)와 선택지 `example`를 추가.
- 모듈에 `wizard: true`면 프런트가 **단계별 위저드**로 렌더(아니면 기존 단일 폼 유지).
- 위저드는 답을 **숨겨둔 기존 입력 칸에 채운 뒤 기존 `generate()` 흐름을 그대로 호출** —
  검증·프롬프트 합성·스트리밍·디자인 주입(PPT) 로직을 재사용. "직접 입력하기"로 기존 폼
  전환 가능(기능 손실 없음).
- API 키 불필요: 위저드·요약·"예시 먼저 보기"(데모)까지 키 없이 동작. 실제 생성만 키 필요.

## 적용 현황

- ✅ 적용: **시험지(exam)**, **자기소개서(cover-letter)**
- ⏳ 미적용(기존 단일 가이드 폼 유지): worksheet, quiz, vocabulary, ppt, study-notes,
  lesson-plan, resume, excel, creative-writing — 아래 제안 질문으로 순차 적용 예정.

## 모듈별 제안 질문 (초안 — 다듬어 주세요)

> 각 질문에는 쉬운 설명·예시·"잘 모르겠어요" 기본값을 함께 답니다.

### 시험지 (exam) — 적용됨
1. 어떤 과목의 시험지를 만들까요? (예: 영어/수학/한국사 · 기본 영어)
2. 어느 범위에서 낼까요? (교재 단원·주제 · 기본 "전체에서 골고루")
3. 난이도는? (하=기초 / 중=보통 / 상=어렵게 · 기본 중)
4. 문제는 몇 개? (보통 20~33개 · 기본 33)
5. 어떤 형태의 문제? (객관식·서술형 · 기본 "골고루")
6. 더 알려줄 점? (선택)

### 자기소개서 (cover-letter) — 적용됨
1. 어떤 일(직무)에 지원하시나요?
2. 자랑하고 싶은 경험은? (필수 — 일터·맡은 일·잘한 점)
3. 왜 이 일을 하고 싶으세요?
4. 본인의 장점은? (예: 성실함/책임감/친절함)
5. 글 길이는? (기본 1000자)
6. 회사가 준 자소서 질문이 있나요? (선택)
7. 지원하는 회사 이름이 있나요? (선택)

### 이력서 (resume)
1. 어떤 일에 지원하시나요? 2. 학교는 어디 나오셨나요?(선택) 3. 어떤 일을 해보셨나요?(경력·활동, 필수) 4. 잘하는 일·자격증이 있나요? 5. 분량은?

### PPT (발표자료)
1. 무엇을 발표하나요?(주제) 2. 누구 앞에서 하나요?(청중) 3. 분량은?(슬라이드 수) 4. 분위기는?(차분/발랄/고급…) 5. 디자인 추천 받기 → 미리보기 → 선택 *(기존 디자인 흐름 연결)*

### 단어장 (vocabulary)
1. 외울 단어를 적어주세요(필수) 2. 누구를 위한 건가요?(수준, 선택) 3. 몇 개로?(선택)

### 수업지도안 (lesson-plan)
1. 어떤 과목인가요? 2. 어떤 단원·주제? 3. 무엇을 가르치고 싶으세요?(학습목표, 선택) 4. 몇 학년/학교급? 5. 수업 시간은?(분)

### 엑셀 (excel)
1. 엑셀로 무엇을 하고 싶으세요?(필수, 예: 월별 합계·차트) 2. 표가 어떻게 생겼나요?(선택) 3. 엑셀인가요 구글 시트인가요?

### 소설·글쓰기 (creative-writing)
1. 어떤 종류의 이야기?(장르) 2. 어떤 이야기인가요?(소재·등장인물, 필수) 3. 분위기는? 4. 길이는?

### 학습지 (worksheet) / 퀴즈 (quiz)
1. 어떤 주제인가요?(필수) 2. 어떤 점을 연습/확인할까요? 3. 난이도는? 4. 문제 수는?
